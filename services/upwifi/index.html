Introduction & Key Benefits

Monitor Guest WiFi performance

Guest WiFi plays an essential role in most venues. Any downtime or interruption in connection can severely disrupt both the guest experience and critical business operations. This is why maintaining the optimal performance of a Guest WiFi service should be a top priority.

UpWiFi is a simple, scalable solution by offering an automated way to monitor your Guest WiFi's key metrics in realtime.

This tool offers an easy-to-use and cost-effective way to track key Guest WiFi parameters, and gain the insights you need for faster WiFi troubleshooting.

Monitor Key Guest WiFi metrics​

Access all metrics via all familiar Zabbix interface.

Receive problem notification to detect and resolve issues faster​

Reduce the frequency and duration of performance bottlenecks and downtime.

Utilize strong Zabbix capabilities​

Enjoy strong data gathering and real-time graphing capabilities of Zabbix.

Key Features

BuWiFi: UpWiFi

Monitor your Guest WiFi's key metrics in realtime

Easy installation

Installing UpWiFi is pretty much plug-and-play.

  • BuWiFi will provide preconfigured Raspberry Pi box to you. It will have your SSID.
  • Place it in-store as close to an Access Point as possible for better measurements.
  • Power it up.
  • And that is it. Preconfigured box will start to take hourly measurements and send it to our Zabbix server.

Measure Guest WiFi key metrics

Get insights from your Guest WiFi network and monitor performance in real-time.

  • Uptime: Amount of time a guest WiFi is active and available for use
  • WiFi Signal Strength (dBm): It can hinge on many different factors, such as the number of devices on the network, background noise and interference, desired data rates, and the types of applications to be used.
  • Throughput (Mbps): The amount of data able to be transferred and received during a specific period of time. Download (DL) and Upload (UL) rates.
  • Latency (ms): Latency is a measurement of delay. The time it takes for a data packet to reach its destination after being sent.

Receive email alerts when something goes wrong.

Receive problem notification to detect and resolve issues faster.

  • Get email notifications from Zabbix when the problem started and resolved.
  • Condition 1: No data collected for 2 hours.
  • Condition 2: Download speed is below limit. If average of last 2 hour download speed goes below average of daily download speed.

Data is provided by all familiar Zabbix web interface

Zabbix is an enterprise-class open source distributed monitoring solution. It has flexible notification mechanism that allows users to configure e-mail based alerts for virtually any event.

  • Strong data gathering and real-time graphing capabilities. Collect data from devices with Zabbix agent.
  • Highly configurable alerting. You can define very flexible problem thresholds, called triggers.
  • Gather and analyze accurate statistics and performance metrics, visualize it, get notified about current and potential issues without delay.
